# AirGap.it Ethereum signer

A pure javascript ethereum signer client side offline app. Easy to review. Uses on purpose CSS only framework (bulma). Gets build with browserify.

# Install
`npm install`

# Build
`npm run build`

# Develop
`npm run watch`
